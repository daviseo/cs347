HTML forms

HTML forms are the easiest way for a user to communicate with the website and access further features or options. Forms add the capability of entering data, which can sent to a server to interpret and store, or modify some part of the visible webpage. For example, adding an item to a list, showing or hiding a feature, and more.

A web form's HTML consists of certian controls often refered to as widgets, as well as elements that give structure to the form. These controls include buttons, text fields, radio buttons, dropdown boxes, checkboxes, and more. These features are mainly achieved through the HTML input tag, although there are other elements that serve more complicated needs.

Here is the code for the radio buttons shown above:
